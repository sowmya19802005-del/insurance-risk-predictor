# Insurance Risk Predictor

A Flask-based web application that calculates a user's insurance risk score and estimated premium based on key factors such as age, income, smoking habits, vehicle type, and past claims.

---

## Features

- Risk score calculation (0–100)
- Dynamic premium estimation
- Input validation with error handling
- Risk category classification (Low / Medium / High)
- UI with detailed breakdown of factors
- Recommendation system based on risk profile

---

## How It Works

The system evaluates the following inputs:

- Age
- Income
- Smoking status
- Vehicle type
- Past claims

Each factor contributes to a total risk score, which is then used to calculate the insurance premium.

---

## Tech Stack

- Backend: Python (Flask)
- Frontend: HTML, CSS, Jinja2
- Version Control: Git, GitHub
